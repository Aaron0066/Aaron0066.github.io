Project Overview

Application Purpose

This project is a custom-built web application for a local fitness coach, aiming to provide interactive scheduling, class details, and health tips for clients. The website will help promote the coachâ€™s services, streamline communication with current and potential clients, and provide a more professional web presence.

Intended Users

  • People looking for a personal trainer
  • Current fitness clients who want access to schedules and resources
  • Visitors interested in fitness advice and healthy lifestyle tips

Website Content Overview

  • Home: Overview and welcome message
  • About: Info on the trainer and their philosophy
  • Services: Classes offered, pricing, and benefits
  • Schedule: Interactive calendar for session sign-ups
  • Contact: Contact form and location info

Site Map

Page Design Details

  • Home: Intro to site and CTA to book sessions. Audience: All. Includes nav, hero image, and CTA button.
  • About: Bio, experience, and certifications. Audience: Prospective clients. Static content only.
  • Services: List of training packages. May include pricing table. Includes buttons to book.
  • Schedule: Interactive calendar with available slots. JavaScript validation for input. Audience: current/potential clients.
  • Contact: Email form and embedded Google Map. Form validations required for name/email/message.

Dynamic Functionality

The project will use JavaScript and jQuery to add the following dynamic features:

  • Interactive session calendar (on Schedule page)
  • Form validation using JavaScript (on Contact page)
  • Animated dropdowns for mobile nav (site-wide)
  • Gallery image slider using jQuery plugins (on Services page)
